# mysql-studies
Projeto inicial de banco de dados utilizando MySQL. Criado para praticar comandos básicos de SQL, como criação de tabelas, inserção de dados, consultas e relacionamentos. Este repositório faz parte do meu portfólio de aprendizado em desenvolvimento de bancos de dados.


# MySQL Learning Project 

Este é meu primeiro projeto utilizando MySQL, criado com o objetivo de aprender e praticar os principais comandos de SQL.  
O projeto é simples, mas cobre conceitos fundamentais para quem está começando no mundo dos bancos de dados relacionais.

---

##  Objetivos
- Criar e gerenciar um banco de dados no MySQL
- Manter uma boa pratica na declação de variações 
- Praticar comandos básicos de SQL:
  - `CREATE DATABASE`, `CREATE TABLE`
  - `INSERT INTO`
  - `SELECT`, `WHERE`, `ORDER BY`
  - `UPDATE`, `DELETE`
- Explorar relacionamentos entre tabelas (`JOIN`)
- Entender boas práticas de modelagem de dados

---

##  Estrutura do Projeto
- `scripts/` → Contém os arquivos `.sql` com os comandos utilizados
- `docs/` → Documentação e anotações sobre o aprendizado
- `examples/` → Exemplos de consultas e resultados
